# Google Sheets API Setup — YouTube Tutorial

A beginner walkthrough for connecting a Python script to Google Sheets using OAuth.

## What this covers

1. Create a Google Cloud project
2. Enable the Google Sheets API
3. Configure the OAuth consent screen
4. Download credentials JSON
5. Save credentials to the right location
6. Create a Google Sheet and copy the spreadsheet ID
7. Configure the spreadsheet ID (config.toml or env var)
8. Authorize on first run — browser prompt, token saved locally

## Intended video length

10–15 minutes, screen-share walkthrough of the Google Cloud Console.
