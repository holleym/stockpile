# CLAUDE.md

## Purpose

YouTube tutorial project: a standalone walkthrough of setting up Google Sheets API
access for personal Python scripts. Covers creating a Cloud project, enabling the
Sheets API, OAuth consent screen, downloading credentials, and authorizing on first run.

## Source material

The setup steps in this project are based on the guide in the sibling repo:
`../positions/docs/google_sheets_setup.md`

## Target audience

Non-developers who want to use Python scripts that write to Google Sheets but have
never set up Google Cloud credentials before.

## Goal

A short, beginner-friendly video (10–15 min) that someone can follow along with
screen-by-screen — no prior Google Cloud experience required.
